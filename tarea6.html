01 · Introducción

¿Qué es Python?

El lenguaje más popular del mundo para IA, datos y automatización.

Python es un lenguaje de programación de propósito general creado por Guido van Rossum en 1991. Su filosofía es la legibilidad: el código Python se lee casi como inglés, lo que lo convierte en el lenguaje ideal para comenzar a programar.

🤖
Inteligencia Artificial
Es el lenguaje principal de machine learning e IA. ChatGPT, TensorFlow y PyTorch están escritos en Python.
🌐
Desarrollo Web
Con Django y Flask puedes crear sitios web y APIs completas muy rápidamente.
📊
Ciencia de Datos
Analiza datos con Pandas, crea gráficas con Matplotlib y modelos con Scikit-learn.
⚙️
Automatización
Automatiza tareas repetitivas: enviar correos, mover archivos, hacer scraping web.
💡
¿Por qué aprender Python? Es el lenguaje #1 más buscado en ofertas de trabajo tech, tiene la comunidad más grande del mundo y su sintaxis limpia facilita aprender otros lenguajes después.

04 · Almacenamiento

Variables

Guarda y reutiliza información en tu programa.

Una variable es como una caja con nombre donde guardas un valor. En Python no necesitas declarar el tipo — lo detecta automáticamente.

# Variable = nombre = valor
nombre = "Ana" ← guarda texto
edad = 17 ← guarda número entero
altura = 1.65 ← guarda decimal
activo = True ← guarda verdadero/falso
# Usar la variable
print(nombre) → Ana
print(edad + 1) → 18
variables.py
# Asignación simple
nombre  = "Carlos"
edad    = 16
altura  = 1.72
activo  = True

# Múltiple asignación
x, y, z = 1, 2, 3

# Reasignar y actualizar
edad = 17     # ahora vale 17
edad += 1     # edad = edad + 1 = 18

# f-string: forma moderna de combinar texto y variables
print(f"Me llamo {nombre} y tengo {edad} años")

# Ver el tipo de una variable
print(type(nombre))   # <class 'str'>
print(type(edad))     # <class 'int'>
⚠️
Reglas para nombrar variables: Solo letras, números y guión bajo. No puede empezar con número. Es sensible a mayúsculas: nombre ≠ Nombre. Sin tildes ni espacios.

06 · Cálculos

Operadores

Matemáticas, comparaciones y lógica en Python.

operadores.py
# ── ARITMÉTICOS ──
print(10 + 3)    # 13  suma
print(10 - 3)    # 7   resta
print(10 * 3)    # 30  multiplicación
print(10 / 3)    # 3.33 división
print(10 // 3)   # 3   división entera
print(10 % 3)    # 1   módulo (resto)
print(2 ** 8)    # 256 potencia

# ── COMPARACIÓN ──
print(5 == 5)    # True  igual
print(5 != 3)    # True  diferente
print(8 > 3)     # True  mayor
print(5 >= 5)    # True  mayor o igual

# ── LÓGICOS ──
print(True  and True)   # True
print(True  or  False)  # True
print(not   True)       # False

# ── ASIGNACIÓN COMPUESTA ──
x = 10
x += 5    # x = 15
x -= 3    # x = 12
x *= 2    # x = 24
🐍
El módulo % devuelve el resto de la división. 10 % 2 == 0 significa que 10 es par. Muy usado para saber si un número es par o impar.

07 · Texto

Cadenas de texto

Todo lo que necesitas saber sobre strings en Python.

strings.py
# f-strings: forma moderna y recomendada
nombre = "Ana"
edad   = 17
print(f"Me llamo {nombre} y tengo {edad} años")
print(f"El doble: {edad * 2}")   # calcula dentro

# Métodos de strings
texto = "  Cultura Digital  "
print(texto.upper())          # CULTURA DIGITAL
print(texto.lower())          # cultura digital
print(texto.strip())          # "Cultura Digital"
print(texto.replace("Digital", "IA"))
print(texto.split())          # ['Cultura', 'Digital']
print(len(texto))             # longitud

# Acceso por índice
palabra = "Python"
print(palabra[0])    # P  (primer carácter)
print(palabra[-1])   # n  (último carácter)
print(palabra[0:3])  # Pyt  (slicing)

# Verificaciones
print("py" in "python")   # True
print("42".isdigit())      # True
💡
Usa siempre f-strings para combinar texto y variables: f"Hola {nombre}". Es más legible que concatenar con + y puedes hacer cálculos dentro de las llaves.

Listas

Colecciones ordenadas y modificables — las más usadas en Python.

listas.py
# Crear listas
frutas  = ["manzana", "pera", "naranja"]
numeros = [1, 2, 3, 4, 5]

# Acceder por índice (empieza en 0)
print(frutas[0])    # manzana
print(frutas[-1])   # naranja (último)
print(frutas[1:3])  # ['pera', 'naranja']

# Modificar
frutas[0] = "mango"      # reemplaza
frutas.append("uva")     # agrega al final
frutas.insert(1, "kiwi") # inserta en posición
frutas.remove("pera")    # elimina por valor
frutas.pop()            # elimina el último

# Información y utilidades
print(len(frutas))          # cantidad
print("uva" in frutas)      # True/False
print(sum(numeros))         # suma todos
print(max(numeros))         # el mayor
print(min(numeros))         # el menor
numeros.sort()              # ordena ascendente
numeros.sort(reverse=True) # ordena descendente

09 · Colecciones

Tuplas y Sets

Colecciones inmutables y sin duplicados.

tuplas_sets.py
# TUPLA — inmutable (no se puede modificar)
colores = ("rojo", "azul", "verde")
punto   = (10.5, -3.2)

print(colores[0])     # rojo
x, y = punto          # desempaquetar
print(x, y)           # 10.5  -3.2

# SET — sin duplicados, sin orden
numeros = {1, 2, 3, 2, 1}   # → {1, 2, 3}
numeros.add(4)
numeros.remove(2)

# Operaciones de conjuntos
a = {1, 2, 3}
b = {2, 3, 4}
print(a & b)   # {2, 3}      intersección
print(a | b)   # {1,2,3,4}   unión
print(a - b)   # {1}         diferencia
🔒
¿Cuándo usar tupla?
Cuando los datos no deben cambiar: coordenadas, colores de marca, días de la semana. Son más rápidas que las listas.
🔂
¿Cuándo usar set?
Cuando necesitas eliminar duplicados automáticamente o hacer operaciones matemáticas de conjuntos.

10 · Colecciones

Diccionarios

Pares clave–valor para organizar información estructurada.

diccionarios.py
# Crear diccionario
alumno = {
    "nombre": "María",
    "edad":   16,
    "grado":  "2° semestre",
    "activo": True
}

# Acceder
print(alumno["nombre"])           # María
print(alumno.get("edad"))         # 16
print(alumno.get("email", "N/A")) # N/A

# Modificar y agregar
alumno["edad"]  = 17
alumno["email"] = "maria@mail.com"

# Eliminar
del alumno["activo"]

# Recorrer con for
for clave, valor in alumno.items():
    print(f"{clave}: {valor}")

print("nombre" in alumno)  # True
print(len(alumno))          # número de claves

11 · Decisiones

Condicionales

Haz que tu programa tome decisiones con if, elif y else.

# Flujo de un if-elif-else
if condición_1: ← si se cumple...
ejecuta este bloque
elif condición_2: ← si no, ¿se cumple esto?
ejecuta este bloque
else: ← si ninguna se cumplió
ejecuta bloque por defecto
condicionales.py
# Ejemplo básico
edad = 17
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

# Múltiples condiciones
calificacion = 8.5
if calificacion >= 9:
    print("Excelente 🌟")
elif calificacion >= 8:
    print("Muy bien ✅")
elif calificacion >= 6:
    print("Aprobado 👍")
else:
    print("Reprobado ❌")

# Condiciones compuestas
usuario  = "admin"
password = "1234"
if usuario == "admin" and password == "1234":
    print("Acceso concedido 🔓")
else:
    print("Acceso denegado 🔒")

# Condicional en una línea (ternario)
estado = "mayor" if edad >= 18 else "menor"
⚠️
¡La indentación es obligatoria! En Python los bloques se definen con 4 espacios. Sin indentación obtendrás un IndentationError. VS Code lo hace automáticamente al presionar Enter después de los dos puntos.

12 · Repetición

Bucles

Repite acciones automáticamente con for y while.

bucles.py
# ── FOR — recorre una secuencia ──
frutas = ["manzana", "pera", "naranja"]
for fruta in frutas:
    print(f"Fruta: {fruta}")

# range() — genera secuencia numérica
for i in range(5):           # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 11):       # 1 al 10
    print(f"{i} x 2 = {i*2}")

# enumerate: índice + valor
for i, fruta in enumerate(frutas):
    print(f"{i}: {fruta}")


# ── WHILE — mientras se cumpla ──
contador = 0
while contador < 5:
    print(f"Contador: {contador}")
    contador += 1  # ¡sin esto es infinito!


# break — sale del bucle
for i in range(10):
    if i == 5: break
    print(i)

# continue — salta la iteración actual
for i in range(10):
    if i % 2 == 0: continue  # salta pares
    print(i)    # imprime solo impares
⚠️
Bucle infinito: Si usas while sin incrementar la variable de control, tu programa se congela. Siempre asegúrate de que la condición eventualmente sea False.

13 · Reutilización

Funciones

Bloques de código reutilizables definidos con def.

funciones.py
# Función básica
def saludar():
    print("¡Hola desde la función!")

saludar()   # llamar
saludar()   # reutilizar

# Con parámetros
def saludar_a(nombre):
    print(f"¡Hola, {nombre}!")

saludar_a("Ana")
saludar_a("Luis")

# Con valor por defecto
def potencia(base, exponente=2):
    return base ** exponente

print(potencia(5))       # 25
print(potencia(2, 10))   # 1024

# Retornar valor
def calcular_promedio(notas):
    return sum(notas) / len(notas)

mis_notas = [8.5, 9.0, 7.5, 10.0]
promedio  = calcular_promedio(mis_notas)
print(f"Promedio: {promedio}")   # 8.75

# Lambda — función de una línea
cuadrado = lambda x: x ** 2
print(cuadrado(8))   # 64
✅
Regla de oro: Una función debe hacer una sola cosa y hacerla bien. Si hace muchas cosas, divídela en varias funciones más pequeñas.

Buenas prácticas

El código limpio separa al principiante del programador.

✅Usa nombres descriptivos: promedio_notas no x. El código se lee más que se escribe.
✅Usa 4 espacios de indentación. VS Code lo hace automáticamente al presionar Enter después de los dos puntos.
✅Agrega comentarios en las partes complejas: # calcula el promedio ponderado
✅Usa f-strings: f"Hola {nombre}" en lugar de "Hola " + nombre
✅Valida los datos del usuario: verifica que input() sea del tipo esperado antes de usarlo.
✅Divide el código en funciones pequeñas — cada una hace una sola cosa.
✅Sigue PEP 8: nombres en snake_case, constantes en MAYÚSCULAS.
✅Usa try / except para manejar errores y evitar que el programa se caiga.
✅Lee los mensajes de error completos — te dicen exactamente qué falló y en qué línea.
❌No uses variables de una sola letra salvo en bucles (i) o coordenadas (x, y).
❌No repitas código — si copias y pegas más de 2 veces, crea una función.
❌No ignores los errores — léelos y entiéndelos.
buenas_practicas.py
# Manejo de errores con try/except
try:
    numero    = int(input("Escribe un número: "))
    resultado = 100 / numero
    print(f"Resultado: {resultado}")
except ValueError:
    print("❌ Debes escribir un número")
except ZeroDivisionError:
    print("❌ No puedes dividir entre cero")
finally:
    print("Programa terminado")


# Código limpio vs. código sucio

# ❌ Sucio
def f(x,y): return x+y+10

# ✅ Limpio
def calcular_total_con_impuesto(precio, impuesto=10):
    """Retorna el precio con el impuesto incluido."""
    return precio + impuesto
