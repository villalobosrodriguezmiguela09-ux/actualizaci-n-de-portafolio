// 01 · Desarrollo Web

Los lenguajes de desarrollo web son las herramientas con las que se construyen sitios y aplicaciones en internet. Se dividen en dos grandes grupos: front-end (lo que el usuario ve) y back-end (la lógica del servidor). Conocerlos es fundamental en la era digital, ya que prácticamente todo lo que usamos en internet está construido con ellos.

01
HTML5 Web · Front-end

Lenguaje de marcado que define la estructura y contenido de una página web. Es la base de todo sitio: sin HTML no hay web. Usa etiquetas para organizar textos, imágenes, enlaces y formularios.

html 
<!-- Estructura básica HTML5 -->
<!DOCTYPE html>
<html lang="es">
  <head>
    <title>Mi página</title>
  </head>
  <body>
    <h1>¡Hola, mundo!</h1>
    <p>Este es mi primer sitio.</p>
    <a href="#">Enlace</a>
  </body>
</html>
Usado en: todos los sitios web del mundo.
02
CSS3 Web · Estilos

Lenguaje de estilos que controla la apariencia visual de las páginas web: colores, tipografías, animaciones y diseño responsivo. Es el "maquillaje" del HTML.

css 
/* Estilos básicos con variables */
:root {
  --azul: #4a90c4;
  --rojo: #c44a4a;
}
.tarjeta {
  background: white;
  border-radius: 12px;
  padding: 1.5rem;
  box-shadow: 0 4px 20px rgba(0,0,0,.1);
  transition: transform .3s ease;
}
.tarjeta:hover {
  transform: translateY(-6px);
}
Usado en: diseño de todas las páginas web.
03
JavaScript Web · Lógica

Lenguaje de programación que añade interactividad y lógica a las páginas web. Con JS se crean menús animados, validaciones, peticiones a servidores y aplicaciones completas (React, Vue, Node.js).

javascript 
// Función de bienvenida con ES6+
const saludar = (nombre) => {
  const hora = new Date().getHours();
  const saludo = hora < 12
    ? "Buenos días"
    : hora < 18 ? "Buenas tardes"
    : "Buenas noches";
  return `${saludo}, ${nombre}!`;
};

console.log(saludar("Tim"));
Usado en: todas las apps web interactivas.
04
PHP Web · Back-end

Lenguaje de servidor que procesa datos del lado del servidor. Se usa en sistemas de login, bases de datos y CMS como WordPress. El 77% de los sitios web con server-side usan PHP.

php 
<?php
// Conexión simple y consulta
$nombre = "Cultura Digital";
$año    = date("Y");

function bienvenida($n, $a) {
  return "Bienvenido a {$n} {$a}";
}

echo bienvenida($nombre, $año);
?>
Usado en: WordPress, Laravel, Moodle.
// 02 · Desarrollo de Escritorio

Los lenguajes de desarrollo de escritorio se usan para crear programas que se instalan y ejecutan directamente en una computadora (Windows, macOS, Linux), sin necesitar internet. Son la base de aplicaciones como editores de texto, reproductores de música y herramientas de productividad.

05
Python Escritorio · IA

Lenguaje de propósito general con sintaxis sencilla. Es líder en Inteligencia Artificial, ciencia de datos y automatización. También se usa para crear apps de escritorio con librerías como Tkinter o PyQt.

python 
# Clase con herencia
class Lenguaje:
  def __init__(self, nombre, año):
    self.nombre = nombre
    self.año    = año

  def info(self):
    return f"{self.nombre} ({self.año})"

py = Lenguaje("Python", 1991)
print(py.info())
Usado en: Netflix, Instagram, NASA, IA/ML.
06
Java Escritorio · Android

Lenguaje orientado a objetos con el lema "Write Once, Run Anywhere". Usado en aplicaciones empresariales, Android y sistemas bancarios. Es uno de los lenguajes más populares del mundo.

java 
// Clase principal en Java
public class HolaMundo {

  public static void main
      (String[] args) {

    String nombre = "Cultura Digital";
    int año = 2025;

    System.out.println(
      nombre + " " + año
    );
  }
}
Usado en: Android, bancos, Minecraft.
07
C# Escritorio · Juegos

Lenguaje de Microsoft para el ecosistema .NET. Es el lenguaje principal de Unity (motor de videojuegos), apps de Windows y desarrollo empresarial. Combina potencia con sintaxis moderna.

c# 
// Clase en C# con propiedades
namespace CulturaDigital;

class Persona {
  public string Nombre { get; set; }
  public int    Edad   { get; set; }

  public string Saludo() =>
    $"Hola, soy {Nombre}";
}
var p = new Persona
  { Nombre = "Ana", Edad = 20 };
Console.WriteLine(p.Saludo());
Usado en: Unity, Visual Studio, Xbox.
08
Swift Escritorio · iOS

Lenguaje de Apple para desarrollar apps en iPhone, iPad y macOS. Fue diseñado para ser rápido, seguro y fácil de leer. Es el sucesor de Objective-C y el estándar actual del ecosistema Apple.

swift 
// Struct y protocolo en Swift
struct Estudiante {
  let nombre: String
  var calificacion: Double

  var aprobado: Bool {
    calificacion >= 6.0
  }
}
let est = Estudiante(
  nombre: "Carlos",
  calificacion: 8.5
)
print(est.aprobado) // true
Usado en: todas las apps del App Store.
// 03 · Diseño & Maquetación

Los lenguajes y herramientas de diseño/maquetación permiten controlar la presentación visual de contenidos digitales e impresos. Van desde hojas de estilo avanzadas como Sass, hasta sistemas de maquetación como Bootstrap y herramientas profesionales como Figma. Son esenciales para cualquier diseñador digital.

09
Sass / SCSS Estilos Avanzados

Preprocesador de CSS que añade variables, mixins, anidado y funciones al CSS tradicional. Hace los estilos más organizados, reutilizables y mantenibles. Se compila a CSS normal para el navegador.

scss 
// Variables y mixin en SCSS
$primario: #4a90c4;
$radio:    12px;

@mixin tarjeta($color: white) {
  background: $color;
  border-radius: $radio;
  padding: 1.5rem;
  &:hover {
    transform: translateY(-4px);
  }
}

.card { @include tarjeta(); }
.card-azul {
  @include tarjeta($primario);
}
Usado en: Bootstrap, grandes proyectos web.
10
Bootstrap Framework CSS

El framework CSS más popular del mundo. Proporciona un sistema de grillas responsivo, componentes listos (botones, formularios, navbars) y utilidades para construir sitios rápidamente sin escribir CSS desde cero.

html + bootstrap 
<!-- Grid de 3 columnas -->
<div class="container">
 <div class="row g-3">
  <div class="col-md-4">
   <div class="card shadow">
    <div class="card-body">
     <h5>Columna A</h5>
    </div>
   </div>
  </div>
 </div>
</div>
Usado en: millones de sitios y proyectos.
11
Figma Diseño UI/UX

Herramienta de diseño colaborativo basada en la nube para crear interfaces, prototipos y sistemas de diseño. No es un lenguaje de código, pero sí un estándar de la industria. Genera CSS automáticamente de sus diseños.

css generado por figma 
/* CSS exportado desde Figma */
.boton-primario {
  display: flex;
  flex-direction: row;
  align-items: center;
  padding: 12px 24px;
  gap: 8px;
  background: #4a90c4;
  border-radius: 999px;
  font-family: 'DM Sans';
  font-weight: 600;
  color: #FFFFFF;
}
Usado en: Google, Airbnb, Spotify, Meta.
12
Tailwind CSS Utility-first CSS

Framework de CSS con clases utilitarias de bajo nivel. En lugar de escribir CSS, aplicas clases directamente en el HTML. Altamente personalizable, muy popular en proyectos modernos con React o Next.js.

html + tailwind 
<!-- Tarjeta con clases Tailwind -->
<div class="bg-white rounded-2xl
  p-6 shadow-lg hover:-translate-y-2
  transition-transform duration-300
  border border-gray-100">
  <h2 class="text-xl font-bold
    text-slate-800">
    Tailwind CSS
  </h2>
  <p class="text-gray-500 mt-2">
    Utility-first CSS
  </p>
</div>
Usado en: Vercel, GitHub Docs, Loom.

✦ Lo que aprendí con esta tarea

Estudiar los lenguajes de diseño y programación me permitió entender que el mundo digital se construye con capas: HTML pone la estructura, CSS la apariencia, JavaScript la vida. Por otro lado, los lenguajes de escritorio como Python, Java, C# y Swift demuestran que la programación va mucho más allá del navegador. Finalmente, las herramientas de diseño como Sass, Bootstrap, Tailwind y Figma revelan que diseñar no es solo tener buen gusto, sino también manejar sistemas y lenguajes especializados. Todos estos lenguajes son parte de la cultura digital que nos rodea cada día.
